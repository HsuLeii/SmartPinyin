Before starting the test, there are a few things to notice:

•

Please do the test with the Chrome browser on the computer.

•

The whole test is an online test, which lasts about 50 to 60 minutes.

•

In the test, each question will be played twice automatically. The system will automatically proceed to the next question soon.

•

When you complete a question group and go to the next one, you cannot return to the previous question group to answer again.

•

If the test is interrupted due to Internet connectivity issues or other factors, the questions answered last time will be continued the next time you log in.

•

If you have any questions or suggestions about the test, please send an e-mail to us: service@coolchinese.net
